# exposly.de

## Exposè Generator für Immobilien

Erstelle professionelle Exposés für:
- 🏠 ImmoScout24
- 📦 eBay Kleinanzeigen
- 📄 PDF-Export
- 🎨 Individuelles Branding

## Features

✅ Benutzerregistrierung & Login
✅ Exposé-Vorlagen
✅ Multi-Plattform Export
✅ Stripe-Zahlungsintegration
✅ Eigenes Branding/Logo
✅ Bild-Upload & Optimierung
✅ Responsive Design

## Schnellstart

```bash
# Repository klonen
git clone git@github.com:Mindlesssloth/Detlef.git
cd Detlef/exposly.de

# Abhängigkeiten installieren
npm install

# Umgebungsvariablen kopieren
cp .env.example .env
# .env Datei mit deinen Werten bearbeiten

# Server starten
npm run dev
```

## Umgebungsvariablen

Siehe `.env.example` für alle benötigten Variablen.

## Tech Stack

- **Backend:** Node.js, Express.js
- **Datenbank:** MongoDB (Mongoose)
- **Auth:** Passport.js, JWT
- **Zahlungen:** Stripe
- **PDF-Generierung:** Puppeteer
- **Frontend:** EJS, Vanilla JS
- **Styling:** Bootstrap 5
